# 📚 Signaalwoorden & Functiewoorden Trainer

Een interactieve web-app om Engelse signaalwoorden en functiewoorden te leren en oefenen!

## 🚀 Live App

[**Open de app op GitHub Pages**](https://niels2-max.github.io/Engels-Signaal--en-Functiewoorden/)

## ✨ Functies

### 📖 Leren
- **Flashcards-modus**: Leer signaalwoorden met interactieve kaarten
- **7 Categorieën**:
  - 🔄 Tegenstelling (however, although, etc.)
  - ❓ Reden / Oorzaak (because, since, etc.)
  - ➡️ Gevolg / Conclusie (therefore, consequently, etc.)
  - ⏱️ Tijd / Volgorde (first, then, finally, etc.)
  - ⚙️ Voorwaarde (if, unless, etc.)
  - 💡 Voorbeelden (for example, such as, etc.)
  - ➕ Toevoeging (moreover, furthermore, etc.)

### ✏️ Oefenen
- **10-vragen Quiz**: Test je kennis
- **Instant feedback**: Zie direct of je antwoord correct is
- **Score tracking**: Krijg je eindresultaat met percentage
- **Moeilijke woorden repeteren**: Perfect voor examenvoorbereiding

## 📊 Dataset

Bevat **30 signaalwoorden** verdeeld over 7 categorieën:

**Tegenstelling** (4): however, although, while, in contrast, on the other hand, yet
**Reden/Oorzaak** (5): because, on account of, since, as, due to
**Gevolg/Conclusie** (4): therefore, as a result, consequently, as a consequence, in conclusion
**Tijd/Volgorde** (5): first, secondly, next, then, finally
**Voorwaarde** (4): if, provided that, unless, in case
**Voorbeelden** (2): for example, such as
**Toevoeging** (3): moreover, furthermore, in addition

## 🎯 Hoe Te Gebruiken

1. **Ga naar de live app**: https://niels2-max.github.io/Engels-Signaal--en-Functiewoorden/
2. **Kies een modus**:
   - **Leren**: Selecteer een categorie en klik "Start Flashcards"
   - **Oefenen**: Klik "Start Quiz" voor een snelle test
3. **Bij Flashcards**: Klik op de kaart om het antwoord te zien
4. **Bij Quiz**: Kies het juiste Nederlands woord
